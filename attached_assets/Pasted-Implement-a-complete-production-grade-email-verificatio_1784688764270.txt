Implement a complete production-grade email verification and transactional email system for Wexora Global using Resend.

Requirements:

## 1. Use Resend
- Integrate the official Resend SDK.
- Use an environment variable:
  RESEND_API_KEY=
- All emails must be sent from:
  Wexora Global <noreply@mail.wexoraglobal.com>

Future addresses:
support@mail.wexoraglobal.com
security@mail.wexoraglobal.com
billing@mail.wexoraglobal.com

## 2. Registration Flow

Current registration must change to:

User enters:
- Name
- Email
- Password

↓

Create a 6-digit verification code.

↓

Store:
- verification code
- expiration time (10 minutes)
- email verified = false

↓

Send beautiful verification email using Resend.

↓

Redirect user to Verify Email page.

↓

User enters code.

↓

If correct:
- mark email_verified=true
- delete verification code
- allow login

If incorrect:
- show error.

If expired:
- show expired message.

## 3. Resend Verification Code

Add:
- Resend Code button
- 60 second cooldown
- Generate a new code.
- Invalidate previous code.

## 4. Login Protection

Prevent login if:

email_verified=false

Return:

"Please verify your email before signing in."

Include:
Resend Verification Email button.

## 5. Forgot Password

Replace current flow.

User enters email.

↓

Generate secure reset token.

↓

Expiry 30 minutes.

↓

Email reset link.

↓

Allow password change.

↓

Invalidate token after use.
